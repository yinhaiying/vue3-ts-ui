# 安装

## npm 安装
推荐使用npm的方式安装，它能更好地和`webpack`打包工具配合使用
```javascript
npm i sea-ui -S
```

## yarn 安装
您也可以使用yarn的方式安装。
```javascript
yarn add sea-ui -S
```

## CDN
未来将会支持通过cdn获取到最新版本的资源，在页面中引入js和css文件即可使用。
```javascript
<!-- 引入样式 -->
<link rel="stylesheet" href="xxx.css">
<!-- 引入组件库 -->
<script src="xxx/index.js"></script>
```
